# FitLife - React Bootstrap Landing Page

FitLife is a responsive single-page landing page built with **React.js**, **Bootstrap 5**, and **Bootstrap Icons**. This project introduces students to component-based development by transforming a traditional HTML Bootstrap application into a modern React application using reusable components.

The application represents a fictional online fitness platform that promotes personalized workout programs and healthy living services. Users can explore the available fitness programs through responsive Bootstrap cards and interact with the interface using basic React event handling.

The project focuses on fundamental React concepts such as functional components, component composition, props, rendering lists with the `map()` method, and handling events with `onClick`. At the same time, students reinforce their Bootstrap knowledge by working with responsive layouts, navigation bars, cards, buttons, spacing utilities, typography, colors, and the Bootstrap Grid System.

## Features

- Responsive Bootstrap navigation bar
- Hero section with a call-to-action button
- Services section generated from external data
- Reusable React components
- Responsive Bootstrap card layout
- Bootstrap Icons integration
- Footer with company information
- Responsive design for mobile, tablet, and desktop devices
- Clean and modular project structure

## Technologies

- React.js
- Vite
- JavaScript (ES Modules)
- Bootstrap 5
- Bootstrap Icons
- CSS3

## Learning Objectives

By completing this project, students will learn how to:

- Create React applications with Vite
- Organize applications using reusable components
- Pass data between components using props
- Render collections using the `map()` method
- Handle user events with React
- Integrate Bootstrap into a React project
- Build responsive user interfaces using the Bootstrap Grid System
- Separate application data from presentation logic
- Structure a professional front-end application

This project serves as an excellent introduction to combining **React.js** and **Bootstrap**, providing a solid foundation for more advanced topics such as state management with Hooks, React Router, forms, API consumption, and full-stack web application development.
